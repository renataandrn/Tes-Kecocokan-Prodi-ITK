## Setup Environment
Sebelum memulai, pastikan telah menginstall beberapa library berikut:
- matplotlib
- numpy
- pandas
- streamlit
Anda dapat menginstallnya di terminal menggunakan `pip`:
```
pip install matplotlib numpy pandas  streamlit
```
## Run streamlit app
```
streamlit run prototype.py
```
